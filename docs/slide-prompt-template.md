## 要件

### 形式
- HTML + CSS（1つのHTMLファイルにまとめる）
- 各スライドは<section class="slide">で囲む
- レスポンシブデザイン（スマートフォン対応）
- スライド間の切り替えはCSSのみ（JavaScript不要）

### デザイン
- フォント：Noto Sans JP（Google Fonts）
- カラースキーム：
  - メインカラー：#2563eb（青）
  - 背景色：#ffffff（白）
  - テキスト：#1f2937（ダークグレー）
  - アクセント：#f59e0b（オレンジ）
- フォントサイズ：
  - タイトル：2rem（32px）
  - サブタイトル：1.5rem（24px）
  - 本文：1rem（16px）
- 余白：各セクションに適切なpadding

### スライド構成
1. タイトルスライド（表紙）
   - サイト名：秋田南くらし相談室
   - 記事タイトル
   - 著者名
   - 日付

2. 目次スライド
   - 記事の主要な見出し一覧

3. 本文スライド（各見出しごと）
   - 見出し
   - 箇条書きや番号付きリスト
   - 重要なポイントは強調表示

4. まとめスライド
   - 記事の要点を3〜5つにまとめる

5. CTA（行動喚起）スライド
   - 「もっと詳しく知りたい方はこちら」
   - LINEで相談するボタン
   - サイトURL

### 挿絵について
- 挿挿絵が必要な箇所には、以下のプレースホルダーを配置してください：
  <img src="/slides/[スライド番号]-[説明].webp" alt="[代替テキスト]" loading="lazy" />
- 例：<img src="/slides/slide-03-smartphone-app.webp" alt="スマホアプリの画面" loading="lazy" />
- 挿絵の説明をAlt属性に記載してください

### アクセシビリティ
- すべての画像にalt属性を付ける
- 色のコントラスト比を4.5:1以上に保つ
- フォントサイズは16px以上

# スマホの基本操作ガイド｜初心者向け

スマホを始めたばかりの方、操作に慣れていない方のために、基本的な使い方をまとめました。

## メールの送受信

### Gmailの基本的な使い方

1. **メールアプリを開く**：画面の「Gmail」アイコンをタップ
2. **メールを読む**：受信トレイに届いているメールをタップ
3. **返信する**：メールを開いて下部の「返信」をタップ
4. **新規作成**：右下の「作成」ボタンで新しいメールを作成

### トラブルシューティング

- メールが届かない場合は、迷惑メールフォルダを確認
- 送信できない場合は、Wi-Fiまたはモバイルデータ通信の接続を確認

## 写真の整理

### 写真の撮り方

1. **カメラアプリを開く**：画面の「カメラ」アイコンをタップ
2. **シャッターボタンを押す**：画面下部の大きな丸いボタン
3. **撮った写真を確認**：左下のサムネイルをタップ

### 写真の整理方法

- **アルバム作成**：写真アプリで「アルバム」タブから作成
- **不要な写真の削除**：写真を長押しして選択→削除
- **写真の共有**：写真を選択→共有ボタンから送信

## LINEの基本操作

### 友だち追加

1. LINEアプリを開く
2. 右上の「友だち追加」アイコンをタップ
3. ID検索またはQRコードで追加

### メッセージの送信

1. 友だちリストから相手を選択
2. 下部のテキストボックスにメッセージを入力
3. 送信ボタン（矢印アイコン）をタップ

## 困ったときは

操作でわからないことがあれば、秋田南くらし相談室にご相談ください。あなたのペースで、ひとつずつお伝えします。

## 挿絵用プロンプト（別途使用）

スライドの挿絵を生成する場合：

```
以下の挿絵をWebP形式で作成してください。

## 挿絵情報
- スライド番号：[番号]
- シーン：[説明]
- サイズ：800px × 600px
- スタイル：フラットデザイン、ミニマル
- カラースキーム：
  - メインカラー：#2563eb（青）
  - 背景色：#ffffff（白）
  - アクセント：#f59e0b（オレンジ）
- キャラクター：なし（アイコンやUI要素のみ）

## 出力形式
- ファイル名：slide-[番号]-[説明].webp
- 解像度：72dpi
- 背景：透明または白
```

## 作成後の手順

1. 生成されたHTMLを `src/pages/slides/[スライド名].astro` に保存
2. 挿絵を `public/slides/` に配置
3. Astroでビルドして確認

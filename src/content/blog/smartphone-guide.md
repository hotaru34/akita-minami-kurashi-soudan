## メールの送受信

### Gmailを開く

1. 画面の下部にある **丸いアイコンが並んでいる場所**（ホーム画面）を探します
2. 「**Gmail**」というアイコンを探してタップします
   - 見つからない場合は、画面を左右にスワイプしてみてください
   - 「Gmail」という文字が書いてある白いアイコンです

### メールを読む

1. Gmailを開くと、受信トレイに届いているメールの一覧が表示されます
2. **読みたいメールをタップ**すると、中身が表示されます
3. 上に戻るには、画面の左上にある **「←」（矢印）** をタップします

### メールに返信する

1. メールを開いて中身を表示します
2. 画面の下部に **「返信」** というボタンがあります
3. 「返信」をタップすると、テキスト入力画面が表示されます
4. 返信の内容を入力します
5. 入力が終わったら、画面の右上にある **「▶」（矢印のボタン）** をタップして送信します

### 新しいメールを作成する

1. Gmailの画面の右下に **「✏️」（鉛筆のアイコン）** があります
2. これをタップすると、新しいメールの作成画面が開きます
3. 「宛先」に相手のメールアドレスを入力します
4. 「件名」にタイトルを入力します
5. 本文を入力します
6. 右上にある **「▶」（矢印のボタン）** をタップして送信します

#### つまずきポイント

- **メールが届かない場合**：画面の左上にある **「≡」（三本線）** をタップ → 「迷惑メール」を確認してみてください
- **送信できない場合**：画面上部の右側に **「📶」や「4G/5G」** の表示があるか確認してください。Wi-Fiか通信が接続されていないと送信できません

## 写真の撮り方と整理

### 写真を撮る

1. 画面の下部にある **カメラアイコン** をタップします（ロック画面からでも起動できます）
2. 画面にカメラの映像が表示されます
3. 撮りたいものを画面の中央に合わせます
4. 画面の下部にある **大きな丸いボタン** をタップします
   - これがシャッターボタンです
5. 撮影されると、画面の左下に **小さなサムネイル（写真の縮小版）** が表示されます
6. サムネイルをタップすると、撮った写真を確認できます

### 写真を見る

1. 画面の下部にある **「写真」アイコン** をタップします
2. 撮った写真が一覧で表示されます
3. **見たい写真をタップ**すると、大きく表示されます
4. 左右にスワイプすると、他の写真も見ることができます

### 写真を整理する

#### アルバムを作成する

1. 「写真」アプリを開きます
2. 画面の下部に **「アルバム」** というタブがあります（または画面の下から3番目のアイコン）
3. 「アルバム」をタップします
4. 画面の右上に **「＋」（プラス）** または **「新しいアルバム」** があります
5. タップしてアルバム名を入力し、写真を選んで追加します

#### 不要な写真を削除する

1. 「写真」アプリで削除したい写真を表示します
2. 写真を **長押し**（指を離さずに1〜2秒おく）します
3. 削除したい写真を選択します（選択するとチェックマークが付きます）
4. 画面の下部または上部にある **「🗑️」（ゴミ箱アイコン）** をタップします
5. 「削除」をタップして確認します

#### 写真を共有する（LINEやメールで送る）

1. 「写真」アプリで送りたい写真を表示します
2. 画面の下部にある **「📤」（共有ボタン）** をタップします
3. 送りたい方法を選択します（LINE、メール、メッセージなど）
4. 相手を選んで送信します

#### つまずきポイント

- **写真が削除できない場合**：写真アプリの「最近削除した項目」フォルダに入っている可能性があります。完全に削除するには、そこから再度削除してください
- **共有ボタンが見つからない場合**：写真をタップしてから、画面の下部を見てください。矢印が上向きのアイコンです

## LINEの基本操作

### LINEをダウンロードする

1. 画面の下部にある **「App Store」**（iPhone）または **「Play ストア」**（Android）アイコンをタップします
2. 画面上部の検索欄に「**LINE**」と入力します
3. 「LINE」というアプリを探して **「入手」**（iPhone）または **「インストール」**（Android）をタップします
4. ダウンロードが完了したら、アプリを開きます

### LINEに登録する

1. LINEアプリを開きます
2. 「新規登録」をタップします
3. 電話番号を入力して「同意して続行」をタップします
4. 届いた認証コードを入力します
5. 名前を入力して登録を完了します

### 友だちを追加する

1. LINEアプリの画面の右上に **「👤＋」（人のアイコン）** があります
2. タップします
3. 「ID検索」を選択して、相手のIDを入力します
4. または「QRコード」を選択して、相手のQRコードを読み取ります
5. 「追加」をタップすると友だちに追加されます

### メッセージを送る

1. LINEアプリの画面の下部に **「トーク」** というタブがあります
2. 「トーク」をタップします
3. 友だちリストから **話したい相手をタップ** します
4. 画面の下部に **テキスト入力欄** があります
5. そこにメッセージを入力します
6. 入力が終わったら、右側の **「▶」（矢印のボタン）** をタップして送信します

### 写真を送る

1. トーク画面（相手との会話画面）を開きます
2. テキスト入力欄の左側に **「＋」（プラス）** ボタンがあります
3. タップして「写真・動画」を選択します
4. 送りたい写真を選んで「送信」をタップします

#### つまずきポイント

- **友だちが追加できない場合**：相手のIDが正しいか確認してください。大文字・小文字も区別されます
- **メッセージが送れない場合**：相手をブロックしている可能性があります。友だちリストで相手の名前を長押しして「ブロック解除」を確認してください
- **LINE電話のかけ方**：トーク画面の右上にある **「📞」（電話アイコン）** をタップすると、LINE電話をかけられます

## よくあるつまずき

### スマホの画面が真っ暗になった場合

1. 電源ボタン（スマホの側面）を **1回押して** 画面を点けます
2. それでも点かない場合は、電源ボタンを **5秒以上長押し** して再起動してください

### アプリが開かない場合

1. 一度スマホを再起動してみてください
2. 電源ボタンを長押し → 「再起動」を選択
3. 再起動後、再度アプリを開いてみてください

### 電池の減りが早い場合

1. 画面の明るさを下げてみてください
2. 画面の上部から下にスワイプして、明るさ調整バーを操作します
3. 使っていないアプリを閉じてください（ホームボタンを2回押すと閉じられるアプリが表示されます）

### スマホが重い（動作が遅い）場合

1. 使っているアプリをすべて閉じてください
2. スマホの空き容量を確認してください（設定 → 一般 → ストレージ）
3. 不要なアプリを削除してみてください
